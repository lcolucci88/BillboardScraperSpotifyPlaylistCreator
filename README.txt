Script to scrap Billboard site for a particular date and use that that to create
a playlist with those songs in your own Spotify account.

Most used case: Create a playlist with trending songs for a particular moment in your life,
birth, marriage, etc.

Please note that you will need to register in the Spotify Developers Dashboard with your Spotify account
to get your own id credentials.

To do this:
Loging with your account into https://developer.spotify.com/dashboard
Create an App
Settings/Copy the Client ID and Client Secret into the python Code.
(use environment variables if not worked locally).
